# Domain Framework - Carrossel Instagram para Mentores

## Framework Operacional

### Fase 1: Pesquisa (Raul Radar)
1. Receber foco/tema do usuário
2. Buscar 8-12 notícias relevantes em fontes variadas
3. Filtrar por relevância para mentores/especialistas
4. Rankear top 5 por: relevância + engajamento + dados
5. Apresentar para seleção do usuário

### Fase 2: Ângulos (Carlos Carrossel)
1. Analisar a notícia selecionada
2. Gerar 5 ângulos emocionais distintos (medo, oportunidade, educacional, contrário, inspiracional)
3. Sugerir formato de carrossel para cada ângulo
4. Apresentar para seleção do usuário

### Fase 3: Criação (Carlos Carrossel)
1. Escolher formato de carrossel (Editorial, Listicle, Tutorial, etc.)
2. Estruturar 8-10 slides com headline + supporting text
3. Definir direção visual por slide
4. Escrever legenda com hook + corpo + CTA
5. Criar lista de hashtags
6. Otimizar engajamento e ortografia

### Fase 4: Revisão (Vera Veredito)
1. Avaliar 10 critérios de qualidade (nota 1-5)
2. Emitir veredito APPROVE/REJECT
3. Se REJECT: feedback acionável para correção
4. Se APPROVE: sugestões opcionais de melhoria

## Métricas de Sucesso

| Métrica | Target |
|---------|--------|
| Slides por carrossel | 8-10 |
| Palavras por slide | 40-80 |
| Score médio de revisão | >= 4.0 |
| Hashtags | 5-15 |
| Hook da legenda | Máx 125 chars |
