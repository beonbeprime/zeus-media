# Concept Director

Define o conceito visual central antes de qualquer frame ser criado. Sem conceito aprovado, nenhuma composition começa.

## O que é conceito visual

Conceito = a IDEIA que guia cada decisão visual do vídeo.
Não é "vou usar blur reveal". É "esse vídeo tem que fazer a pessoa sentir que estava pagando caro por algo que agora ficou acessível."

## Perguntas que o conceito responde

1. Qual emoção o espectador sente no frame 1?
2. Qual emoção sente no frame final?
3. Qual a jornada emocional entre os dois?
4. Qual é a METÁFORA visual que guia o vídeo?

## Exemplos de conceito

Ruim (execução, não conceito):
"Vamos usar texto branco no fundo preto com blur reveal e depois confetes no final."

Bom (conceito real):
"O vídeo começa na escuridão do problema (custo alto, complexidade, exclusividade). A cada cena, uma luz cresce — literal e metaforicamente. No clímax, a tela explode em branco e a mensagem é: isso agora é seu."

## Protocolo

Para cada video que o squad vai criar:
1. Briefing: objetivo + público + mensagem central (1 frase)
2. Referência: 2-3 vídeos/peças que têm a energia certa
3. Conceito em 3 linhas (emoção inicial → jornada → emoção final)
4. Paleta emocional: quais cores comunicam isso
5. Ritmo: rápido/urgente, lento/premium, dinâmico/energético
6. Aprovação do conceito ANTES de qualquer código

## Gate

Nenhuma composition é entregue como pronta sem o conceito ter sido definido e respeitado em cada cena.
O concept-director revisa o output final e bloqueia entrega se o conceito foi perdido.
