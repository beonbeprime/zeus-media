# Typography System - Zeus Media

## Funcao
Selecionar e configurar tipografia baseada no estilo visual da pessoa.

## Matriz de Selecao

| Estilo | --font-main | --font-body | Heading weight | Body weight |
|--------|-------------|-------------|----------------|-------------|
| Classico/Elegante | Cormorant Garamond | DM Sans | 300, italic accent | 400/500 |
| Moderno/Futurista | Inter | DM Sans | 700-800 | 400/600 |
| Minimalista | Inter | Inter | 300-400 | 400 |
| Ousado/Vibrante | Space Grotesk | DM Sans | 700-800 | 400/500 |
| Vintage/Retro | Playfair Display | Lora | 400-600 | 400 |

## Google Fonts Links

Classico:
```
family=Cormorant+Garamond:ital,wght@0,300;0,400;0,600;1,300&family=DM+Sans:wght@400;500;600;700
```

Moderno:
```
family=Inter:wght@300;400;500;600;700;800;900&family=DM+Sans:wght@300;400;500;600;700
```

Minimalista:
```
family=Inter:wght@300;400;500;600;700
```

Ousado:
```
family=Space+Grotesk:wght@400;500;600;700&family=DM+Sans:wght@400;500;600;700
```

Vintage:
```

```
